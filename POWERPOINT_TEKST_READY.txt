SLIDE 2: O PROJEKTU
-------------------
Šta je AutoDijelovi Shop?

• Online shop za prodaju autodijelova
• Omogućava korisnicima pregled i kupovinu dijelova
• Admin panel za upravljanje proizvodima
• Multi-language podrška (Bosanski/Engleski)
• Responsive dizajn za sve uređaje

SLIDE 4: BAZA PODATAKA
-------------------
3 Tabele u MySQL Bazi:

1. USERS (Korisnici)
   • id, name, email, password
   • is_admin (Boolean)

2. CATEGORIES (Kategorije)
   • id, name, description
   • created_at, updated_at

3. AUTO_PARTS (Autodijelovi)
   • id, category_id (FK)
   • name, description, price, stock, image
   • created_at, updated_at

SLIDE 7: CRUD - CREATE
-------------------
Dodavanje Novog Dijela

Forma sadrži:
• Naziv dijela
• Kategorija (dropdown)
• Opis (textarea)
• Cijena (KM)
• Stock (količina na lageru)
• Slika (upload)

Validacija:
• Sva polja obavezna
• Slika max 2MB (JPG, PNG, GIF)
• Cijena mora biti pozitivna
• Stock mora biti broj ≥ 0

Pristup: Samo administratori

SLIDE 9: CRUD - UPDATE
-------------------
Izmjena Postojećeg Dijela

Proces:
• Admin klikne "Edit" u tabeli
• Forma se otvara popunjena sa podacima
• Trenutna slika je prikazana
• Admin mijenja željene podatke
• Klik na "Sačuvaj"
• Poruka: "Dio uspješno izmijen"

Napomena:
• Može se promijeniti slika (nova zamjenjuje staru)
• Svi podaci se validiraju


SLIDE 10: CRUD - DELETE
-------------------
Brisanje Dijela

Proces:
1. Admin klikne "Delete" (ikona korpe)
2. JavaScript confirm dialog
   "Da li ste sigurni da želite obrisati ovaj dio?"
3. Ako klikne OK:
   • Dio se briše iz baze
   • Slika se briše sa servera
   • Poruka: "Dio uspješno obrisan"
4. Ako klikne Cancel:
   • Ništa se ne briše

SLIDE 16: STOCK MANAGEMENT
-------------------
Automatsko Upravljanje Zalihama

Kako radi:

1. Početno stanje
   Dio ima stock = 10 kom

2. Korisnik klikne "Kupi"
   Stock se automatski smanji → 9 kom
   Poruka: "Kupovina uspješna!"

3. Kada stock = 0
   Dugme postaje disabled
   Tekst: "Rasprodato"
   Badge: Crvena boja

Admin može:
• Dodati nove zalihe (edit)
• Pratiti stock u tabeli


SLIDE 17: UPLOAD SLIKA
-------------------
Upload i Prikaz Slika

Proces:

1. UPLOAD
   • Input type="file" u formi
   • Validacija: JPG, PNG, GIF
   • Max veličina: 2MB

2. ČUVANJE
   • Storage folder: storage/app/public/parts/
   • Automatski generiše ime

3. PRIKAZ
   • asset('storage/' . $image)
   • Placeholder ako nema slike

4. BRISANJE
   • Pri delete → briše sliku
   • Pri update → zamjenjuje staru

SLIDE 23: INSTALACIJA
-------------------
Kako Pokrenuti Projekat

Preduvjeti:
• XAMPP (Apache + MySQL)
• Composer

Koraci:
1. composer install
2. copy .env.example .env
3. php artisan key:generate
4. Kreiraj bazu: autodijelovi_db
5. Import SQL: autodijelovi_dump.sql
6. php artisan storage:link
7. php artisan serve
8. Otvori: http://localhost:8000

Trajanje instalacije: 5-10 minuta
